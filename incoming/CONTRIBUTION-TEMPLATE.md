# How to Contribute

Drop a file in this folder. We will review it and fold it into the platform.

## Format does not matter

We accept anything:
- A fully drafted skill
- A war story or retrospective note
- A template or checklist
- A half-formed idea
- A bullet list of anti-patterns
- A conversation transcript

We structure it. You just share what you know.

## If you want to help us, include

```
Skill name: [what would you call it?]
Category: [foundational | discovery | delivery | strategy | leadership]
Author: [your name or "anonymous"]
Date: [today]

What situation does this address?
...

What should a PM know or do?
...

Do you have a real example?
...
```

## Rules
- Anonymous submissions are fine.
- One idea per file.
- War stories and real examples are gold.
